### Tech stack used for this app
	- ASP.NET CORE
	- React.js & Redux
	- Styled Components
	- Enzyme

### How to run this app in your local machine
```
> export ASPNETCORE_Environment=Development
> dotnet run
```
Then navigate to http://localhost:5000

### Run tests
```
> cd ClientApp
> npm test
```
